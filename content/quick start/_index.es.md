Comenzar con Cantara es **muy fácil** y se puede hacer siguiendo estos tres pasos:

 1. Descargar Cantara
 2. Recopilar canciones
 3. Ejecutar y configurar Cantara

Mientras que una explicación detallada con enlaces de descarga se puede encontrar en la [sección de Tutorial]({{% ref "/tutorial/_index.en.md" %}}), esta página contiene solo algunos consejos breves.

## Descargar Cantara

Hay varios enlaces de descarga para diferentes sistemas operativos.

* Vaya a la [sección del Tutorial de descarga e instalación]({{% ref "/tutorial/Install Cantara.en.md" %}}) para más información.
* Consulte la [página de Releases en Github](https://github.com/reckel-jm/cantara/releases) para más opciones.

## Recopilar canciones

¡Use servicios como CCLI Song Select u otras bases de datos para obtener letras de las canciones que quiere cantar!
¿No sabe por dónde empezar? [Eche un vistazo aquí]({{% ref "/tutorial/Where to get the songs.en.md" %}}).

## Ejecutar y configurar Cantara

La configuración de Cantara debería ser autoexplicativa para la mayoría de los casos. Si necesita más información, también puede dirigirse a la [sección de tutorial]({{% ref "/tutorial/How to use it.en.md" %}}).
