Cantara 可以轻松地翻译成其他语言。这样做，您将帮助在全球范围内传播该软件的使用，让不会说英语的人也能访问 Cantara。

## 工作原理

Cantara 使用语言文件（`*.mo` 文件），其中包含二进制语言中的已翻译语言字符串。这些文件可以使用 [Poedit](https://poedit.net/) 等编辑器和 [cantara.pot](https://github.com/reckel-jm/cantara/blob/master/locals/cantara.pot) 文件作为模板来创建。

如果您想翻译成新语言，请按以下步骤操作：

1. 下载 [cantara.pot](https://github.com/reckel-jm/cantara/blob/master/locals/cantara.pot) 并用适当的编辑器打开它。
2. 将字符串翻译成所需的目标语言。
3. 将翻译后的内容导出为 `.mo` 文件，**但也请保留翻译后的 `.po` 文件！**
4. 将 `语言.mo` 文件复制到 `<cantara目录>/languages/语言/cantara.mo`（将 `语言` 替换为语言的短代码，例如，*de* 表示德语，*en* 表示英语，*zh* 表示中文等）。
5. 使用 `--lang` 选项运行 Cantara 测试您的翻译：

    `cantara --lang=<语言代码>`


6. 如果翻译运行良好，请在 GitHub 上发起 Pull Request，以便其他人也能使用它。
