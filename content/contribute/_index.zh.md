Cantara 是自由软件，以大量的耐心和热情开发而成。如果您喜欢它，如果您能考虑以下事项，我将不胜感激。感谢您使用 Cantara，这对我来说是一种荣幸！

## 提供反馈、建议和错误报告

如果您发现任何问题，有想法或建议，或想报告错误，请[在 Cantara Github 仓库中开启一个 issue](https://github.com/reckel-jm/cantara/issues)。我完全通过 Github 处理项目事务，所以这是最好的方式。如果您没有 Github 账户且不想创建，请让朋友代劳。

## 将 Cantara 翻译成其他语言

Cantara 被世界各地的人们使用。随着时间的推移，一些志愿者贡献了多种语言的翻译。查看[翻译部分](translate/)了解当前状态和支持我的方式。

## 推荐给他人

使用 Cantara 的人越多，程序就能变得越好！欢迎通过您可用的渠道推荐 Cantara。

## 不要向我捐款，而是捐给其他人

Cantara 的作者拥有全职工作的特权，因此目前不需要也不希望获得财务支持。因此，目前 Cantara 不需要捐款。但是，世界上有很多人目前真正处于困境或痛苦中。如果您喜欢 Cantara，并发现在使用该软件时节省了一些钱，请考虑支持以下我真正欣赏并希望支持的非营利组织：

* [OpenDoors](https://www.opendoors.de/)：一个帮助全世界受迫害基督徒的基督教组织。
* [Missionswerk Werner Heukelbach](https://www.heukelbach.org/)：一个在德国传播福音的宣教组织。

感谢您的帮助！
