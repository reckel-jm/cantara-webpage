Cantara ist ein Freiwilligenprojekt und bietet daher keinen professionellen Support. Der Entwickler hilft jedoch sehr gerne, wenn es Fragen oder Schwierigkeiten gibt. Der bevorzugte Kontaktweg ist das [Öffnen eines GitHub-Issues](https://www.github.com/reckel-jm/cantara/issues).

GitHub ist eine kostenlose Software-Kollaborationsplattform. Sie können dort ganz einfach ein Konto erstellen und dann ein Issue mit Ihrer Frage oder dem Problem öffnen, das Sie haben.
