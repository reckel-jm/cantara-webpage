Die folgenden Seiten geben Ihnen eine detaillierte Einführung in Cantara und wie die Software funktioniert.
Für die meisten Benutzer sollte Cantara von Haus aus gut funktionieren.
Falls es jedoch Probleme oder Schwierigkeiten gibt, die nicht gelöst werden können, besuchen Sie gerne das [Github-Repository](https://github.com/reckel-jm/cantara) und fragen Sie nach zusätzlicher Hilfe.

{{% notice info %}}
Bitte beachten Sie, dass dieses Tutorial für Version 2.5.0 und später gilt. Frühere Versionen unterstützen nicht alle Funktionen, die in diesem Abschnitt beschrieben werden.
{{% /notice %}}

Beim Lesen der folgenden Seiten können Sie lernen, wie Cantara funktioniert, wie Sie es herunterladen können, wo Sie Liedtexte bekommen und wie Sie das Programm verwenden können.

Der Autor freut sich jederzeit über Rückmeldungen und Empfehlungen. Vielen Dank!
